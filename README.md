# opensourceFLOWCAM

opensourceFLOWCAM is a universal spatial-state runtime that converts camera feeds into adaptive probabilistic state graphs.

Using commodity cameras and proven tracking pipelines, FLOWCAM projects observed physics into domain-specific inference modes:

- DEV(TEST)
- WORK
- GAME

One runtime.
Multiple semantic projections.
Real-time adaptive state inference.

---

## Core Pipeline

```text
Camera Feed
→ Tracking
→ Spatial Graph
→ Adaptive Markov Engine
→ Projection Layer
→ State Prediction
```

---
